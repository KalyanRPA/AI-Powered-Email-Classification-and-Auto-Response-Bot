AI-Powered Email Classifier & Auto Response Bot

Project Description

The AI-Powered Email Classifier & Auto Response Bot is an intelligent email automation solution developed using UiPath. The bot automatically reads incoming emails, analyzes their content, and categorizes them into different groups based on their purpose and urgency. Based on this classification, the automation performs actions such as sending automated replies, flagging high-priority emails for human verification, or filtering non-actionable messages.
This automation significantly reduces manual effort involved in email handling. It helps organizations respond faster to customer queries, ensures important emails are prioritized, and improves the overall efficiency of email management.

How It Works

The automation follows a Dispatcher–Performer architecture where incoming emails are first collected and stored as transactions, and then processed individually. This design helps the system handle multiple emails efficiently while maintaining a structured workflow.

Dispatcher

The Dispatcher bot is responsible for collecting incoming emails and adding them to a queue so they can be processed later. The bot reads emails from the mailbox using the For Each Email activity. For every email that is detected, it extracts important details such as 
1.	Sender
2.	Subject
3.	Body content 
4.	Unique mail identifier.
	
After extracting this information, the Dispatcher creates a queue transaction containing the email data. Each email therefore becomes a queue item that can be processed independently by the Performer bot. This ensures that emails are organized as transactions and can be handled in a scalable and reliable manner.

Performer

The Performer bot processes the queue items generated by the Dispatcher. When the automation starts, it first performs initialization where the categories used for email classification are defined. These categories include 

1.	Replyable
2.	Non-Replyable
3.	High Priority


After initialization, the bot retrieves a transaction from the queue in the Get Transaction state. Each transaction represents one email that needs to be analyzed and processed.
During the Process state, the automation first validates the email details such as the body content and the sender and receiver information. If any of these values are missing or invalid, the system throws a business exception and the email is moved to the Defective Mails folder.
The bot then checks whether the subject contains the phrase "no reply". Emails with this phrase are treated as promotional or automated system messages and are moved directly to the Promotional Mails folder.
If the email passes these checks, the system categorizes the email based on its content. Emails that belong to categories such as support requests, queries, orders, service requests, or help requests are treated as 

1.	Replyable emails. For these emails, the automation generates an appropriate reply message, sends the response to the sender, marks the email as read, and moves it to the Processed Mails folder.
Some emails require immediate human attention. Emails related to refunds, payment issues, or system problems are categorized as

2.	High Priority. For these emails, the automation generates an acknowledgement message informing the sender that their request has been received. The acknowledgement email is sent automatically, the original email is marked as unread, and it is moved to the High Priority Mails folder so that a human agent can review it.
Certain emails do not require any response. These include newsletters, promotional content, marketing messages, system notifications, circulars, or general informational updates. Such emails are categorized as

3.	Non-Replyable. For these emails, the automation marks the email as Read and moves it to the Unprocessed Mails folder so that unnecessary emails are filtered and the inbox remains organized.
	
By combining automated email classification with queue-based processing, this solution enables efficient email management. It ensures that important emails receive quick attention while routine or promotional emails are filtered automatically.
